# AI Medical Chatbot (RAG-Based)

AI Medical Chatbot built using LangChain, FAISS, Hugging Face, and Streamlit.

## Features
- Chat with medical PDFs
- Semantic search using FAISS
- RAG pipeline
- Streamlit UI

## Tech Stack
Python, Streamlit, LangChain, FAISS, HuggingFace

## Run Project
pip install -r requirements.txt
streamlit run medichatUI.py
